# my-vue-app

Based on [Vue.js as desktop executable](https://www.ralfderijcke.nl/blog/2020-10-20-vue.js-as-desktop-executable/)

## Project setup
```
npm install
```

```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
